# Project: Basic Network Configuration

## Overview
This lab demonstrates basic device configuration in packet tracer, with a 3650 Switch, a 2911 Router, Console cables and two laptops.

## Topology
![Network Topology](device-config.png)

## Technologies Used
- Terminal connection emulating real world putty connection

## Objectives
1. Establish connection to the a 3650 switch through the laptop terminal over the console cable
2. Establish connection to the 2911 router through the laptop terminal over the console cable
3. Preview initial device booting process
4. Rename the devices and save the running configs
